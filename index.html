DIRECTIONS...

  • Warning: Best viewed on a desktop computer, not mobile-ready.

  • You can paint using detail or stream methods.

  • The default color is #EFEFEF80. This color can be used to erase other colors while in paint mode. Use the color picker box to select colors not already provided by the 15 preset color buttons. (Be sure to hit the elongated color picker button to activate your chosen color.)

  • Switch to "dungeon tile" mode (via the middle toggle switch labeled, "Paint or Dungeon Tile") to create a dungeon map using the 150 tiles preset options provided. Switching will render paint mode hidden.

  • Dungeon tile maps can be painted over by returning to paint mode.

  • Nota bene: the stream button in paint mode requires that you hold down the left click button on the mouse. Sometimes this function "sticks" though, causing a follow up left click to end the stream of color.
